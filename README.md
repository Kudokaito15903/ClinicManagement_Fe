# Clinic Management System - Frontend

Ứng dụng quản lý phòng khám hiện đại được xây dựng với React, TypeScript và Vite.

## Tính Năng Chính

- **Quản Lý Bệnh Nhân**: Thêm, sửa, xóa và xem thông tin bệnh nhân
- **Quản Lý Bác Sĩ**: Quản lý danh sách bác sĩ và lịch làm việc
- **Quản Lý Phòng**: Quản lý các phòng khám và bố trí không gian
- **Quản Lý Đơn Thuốc**: Tạo và in đơn thuốc cho bệnh nhân
- **Quản Lý Dịch Vụ**: Quản lý các dịch vụ y tế cung cấp
- **Quản Lý Chẩn Đoán**: Theo dõi các chẩn đoán và bệnh lý
- **Quản Lý Khám Bệnh**: Xử lý khám bệnh cho bác sĩ và lễ tân
- **Quản Lý Thanh Toán**: Xử lý hóa đơn và thanh toán
- **Báo Cáo**: Xem báo cáo doanh thu và thống kê
- **Bảng Điều Khiển**: Dashboard tổng quan về hoạt động phòng khám

### Yêu Cầu
- Node.js v22 trở lên
- npm hoặc yarn

### Các Bước Cài Đặt

1. **Clone repository**
```bash
git clone <repository-url>
cd ClinicManagement1/Fe
```

2. **Cài đặt dependencies**
```bash
npm install
```

3. **Cấu hình API**
- Chỉnh sửa file `src/api/client.ts` để kết nối đến API server
- Đảm bảo backend API đang chạy

4. **Chạy ứng dụng phát triển**
```bash
npm run dev
```

Ứng dụng sẽ mở tại `http://localhost:5173`
